*Introduction*

A giter8 based quick setup tool for the Lift web framework and helpers.

*Components*

1.  Lift Framework (current stable version 2.4)

- Blank Template

- Basic Template

- HTML5 Boilerplate + Bootstrap 2.0

- HTML5 Boilerplate + Kickstrap

2.  SBT

3.  SBT Extras script

4.  Lifty

5.  giter8

6.  Scala
